Los tres pilares de la Web

Tim Berners-Lee es cornsiderado el padre de la Web porque desarrolló los tres elemntos básicos para el funcionamiento de la Web:

  • HTML
  • HTTP
  • URL

HTML

HTML, siglas en inglés de HyperText Markup Language(‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web.
Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código
(denominado código HTML) para la definición de contenido de una página web.
